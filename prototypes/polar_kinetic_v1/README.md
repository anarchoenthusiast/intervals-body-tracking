# Polar Kinetic Typography Tool (Prototype v1)

Прототип инструмента для генерации кинетической типографики на основе полярных координат.
Используется для создания анимаций в стиле "сканирования", "цветов" и "спиралей" из текста.

## Функционал
- **Генерация узора:**
  - Настройка текста
  - Количество лепестков (Petals)
  - Количество колец (Rings)
  - Закручивание (Twist) — создание спиралей
  - Амплитуда искажения
- **Экспорт:**
  - Запись видео (WebM)
  - Выбор разрешения (1080p, etc)
  - Выбор соотношения сторон (1:1, 9:16, 16:9)
  - **Прозрачный фон** (альфа-канал) для использования в видеоредакторах.

## Как запустить
1. Открыть `index.html` в браузере (желательно через локальный сервер, например `python3 -m http.server`).
2. Использовать панель управления справа (dat.GUI).

## Технический стек
- HTML5 Canvas
- p5.js (рендеринг)
- dat.gui (интерфейс)
- MediaRecorder API (запись видео)

## Известные особенности
- Запись видео требует поддержки кодеков VP9/VP8 в браузере (Chrome/Firefox рекомендуются).
- Для прозрачности используется `clear()` вместо `background()`.
